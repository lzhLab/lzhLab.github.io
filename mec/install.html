Prerequisite

MEC is developed based on FM-index. The BWT is constructed by RopeBWT2 [1], and the FM-index is developed based on the alternative implemented in SGA [2]. The OpenMP [3] is used for parallelization and GCC 4.9 [4] or higher version is required to compile MEC. The google sparsehash is also required.

Installation

  1. Install GCC 4.9 or higher version.
  2. Install OpenMP 1.10.
  3. Download MEC source code, and decompress it.
  4. Install MEC by type:
    5. ./autogen.sh && ./configure && make














Reference

  1. RopeBWT2: Li H., Fast construction of FM-index for long sequence reads, Bioinformatics: 2014, 10.1093/bioinformatics/btu541.
  2. SGA: Simpson J.T. and Durbin R., Efficient de novo assembly of large genomes using compressed data structures, Genome Res.: 22(3):549-56. doi: 10.1101/gr.126953.111.
  3. OpenMP: http://openmp.org/wp/
  4. GCC: https://gcc.gnu.org/
